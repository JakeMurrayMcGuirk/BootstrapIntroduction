Introductory Bootstrap exercise

Open this file in VS Code and complete the following steps, loading the file in a browser after each step to check the effect in the rendered page. You should not add any styling instructions except via Bootstrap classes.

  1. Examine the file and the content structure. You will see eight section HTML elements. Add a border to these elements, by assigning them to the Bootstrap class border.
  2. By exploring Bootstrap 5 documentation on W3 Schools or other source, for each section element find one or more Bootstrap classes that produce the style requested by the element's text and apply them.
  3. Each section element is directly inside a div element. Turn these div elements into medium Bootstrap columns of sizes 2, 3, 2, 3, x, 3, 4, y. Determine x and y so that the first five columns fill the first row and the last three columns fill the second row. The rows are represented by div elements assigned to the Bootstrap class row (confirm this by examining the file).
  4. Get all the section elements in each row to take the full height of the containing column and row, by assigning them to the class h-100.
  5. Give each column element a little space by assigning it to class mt-3.
  6. Add some padding to each section element that doesn't have it already, by using the Bootstrap class p-2.
